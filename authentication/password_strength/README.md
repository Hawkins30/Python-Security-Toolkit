# Password Strength Checker

## Overview

This tool demonstrates a simple password strength checker based on basic input validation rules such as length and character composition.

It is designed to illustrate how applications can enforce minimum password quality requirements, rather than to provide a comprehensive strength assessment.

---

## Security Concept Demonstrated

- Password policy enforcement
- Credential quality validation

---

## What This Tool Shows

- How weak passwords can be detected using simple rules
- Why enforcing password requirements is important for authentication security
- The limitations of naïve password strength checks

---

## Limitations

- Does not use entropy-based or scoring-based analysis
- Does not check for common passwords or breached credentials
- Rules are simplistic and easily bypassed
- Not suitable as a standalone password validation system

This example is intentionally minimal to demonstrate core concepts rather than best-in-class security.

---

## How to Run

```bash
python main.py
```

# 密码强度检测（Password Strength Checker）

## 项目概述

该工具通过一组简单的规则（如长度和字符类型）来演示密码强度检测的基本方法。

本示例用于说明应用程序如何实施最低密码质量要求，而非提供完整或严格的密码强度评估。

---

## 演示的安全概念

- 密码策略与规则校验
- 凭证质量控制

---

## 本工具展示了什么

- 如何使用简单规则识别弱密码
- 为什么密码复杂度对身份认证安全至关重要
- 基础密码强度检测方法的局限性

---

## 限制说明

- 未基于熵或评分模型进行分析
- 未检测常见密码或已泄露密码
- 规则较为简单，容易被绕过
- 不适合作为独立的密码验证系统

该示例旨在演示核心概念，而非提供最佳安全实践。

---

## 运行方式

```bash
python main.py
```
